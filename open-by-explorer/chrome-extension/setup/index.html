Initialization 初期設定

Those steps are required to allow the extension to be fully operational. これらの手順は、拡張機能を完全に機能させるために必要です。

  1. Install Node.js.
    ※ The LTS recommended version is OK.     Node.js をインストール
    ※ LTS 推奨版でOK
  2. Create a folder in a suitable location that will not be deleted.
    ※ If the folder disappears due to deletion or movement, this initial setup needs to be re-enforced.     削除されない適当な場所にフォルダを作成する
    ※ フォルダが削除や移動によって存在しなくなると本初期設定の再実施が必要
  3. Save the following files in the folder you just created. 以下のファイルを作成したフォルダに保存
    • manifest.json
    • native-messaging-host-app.bat
    • native-messaging-host-app.js
  4. Path to the folder you created : 作成したフォルダのpathを以下に入力
  5. Registry file generation, perform registry edition either from manual edition OR from launching the generated file : レジストリ登録
    • (ADVANCED users only) Manual configuration from the registry editor. レジストリエディターから手動で設定
    • (PREFERRED solution) Download the generated Registry registration file and launch it (double-click). レジストリ登録ファイルを ダウンロード して実行
  6. Operation check 動作確認
    1. Add the path to a folder to Open in Explorer Explorerで開く ディレクトリ/ファイル のパス
